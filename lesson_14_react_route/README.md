# React Router, ленивая загрузка, Suspense

1. <b>Цель:</b>
	* Освоить маршрутизацию в React, ленивую загрузку компонентов и глобальное состояние маршрутизации.

1. <b>Описание/Пошаговая инструкция выполнения домашнего задания:</b>
	* Добавить в приложение React маршрутизацию через React Router, 
    * Реализовать ленивую загрузку (lazy/Suspense) для отдельных страниц
    * Хранить глобальное состояние маршрутизации (например, выбранный раздел) в сторе.

1. <b>Задачи:</b>
    * Настроить React Router с минимум 3 маршрутами, включая параметризованный.
    * Добавить lazy() + Suspense() для ленивой загрузки минимум 1–2 страниц.
    * Хранить часть состояния маршрутизации глобально (Redux/Context) и использовать его в компонентах.
