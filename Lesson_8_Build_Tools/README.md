# Настройка сборки проекта (Vite/Webpack/Rollup)

1. <b>Цель:</b>
	* Освоить современный сборщик (Vite/Webpack/Rollup)
    * Настроить сборку HTML/CSS/JS и базовые инструменты качества.

1. <b>Описание/Пошаговая инструкция выполнения домашнего задания:</b>
	* Настроить сборку проекта с использованием современного сборщика. 
    * Проект должен собирать HTML, CSS и JS, поддерживать dev?сервер и иметь минимальную структуру исходников и артефактов.

1. <b>Задачи:</b>
    * Инициализировать проект и подключить сборщик (Vite/Webpack/Rollup) с dev?сервером.
    * Настроить сборку HTML, CSS и JS, вынести исходники в src/, сборку в dist/.
    * Подключить линтеры (ESLint/Prettier) и настроить их запуск в CI.
    * Добавить скрипты npm для dev/build/preview, приложить краткую инструкцию в README.

    
1. <b>Описание scripts в package.json:</b>
    * `npm run build` Сборка проекта для продакшена, файлы которых собираются в dist
    * `npm run dev` Запуск dev?сервера
    * `npm run dev:safe` Запуск dev?сервера только при корректном форматировании кода, eсли Prettier найдёт ошибки — сервер не запустится
    * `npm run preview` Запуск продакшн сервера из папки dist, предварительно сбилженные файлы
    * `npm run lint` Проверка TypeScript?файлов через ESLint, согласно правилам из файла eslint.config.js
    * `npm run dev` Запуск dev?сервера
    * `npm run format` Автоматическое форматирование всех файлов по включенным правилам из .prettierrc
    * `npm run format:check` Проверка соответствия кода правилам Prettier, и если ошибок нет, то возвращает 1 (необходимо для запуска dev:safe)
