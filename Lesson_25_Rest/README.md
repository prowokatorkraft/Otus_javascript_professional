# REST API с CRUD (пример: «Клон LeetCode»)

1. <b>Цель:</b>
	* оосвоить разработку REST API с CRUD‑операциями.

1. <b>Описание/Пошаговая инструкция выполнения домашнего задания:</b>
	* Реализовать REST API для сущностей «задача» и «пользователь»: создание, чтение, обновление, удаление. 
	* Предусмотреть пагинацию и фильтрацию списка задач.

1. <b>Задачи:</b>
    * Инициализировать серверный проект (Node/Nest/Express — на выбор).
	* Реализовать CRUD для задач и пользователей, продумать DTO/валидацию.
	* Добавить базовые тесты для контроллеров/сервисов.
