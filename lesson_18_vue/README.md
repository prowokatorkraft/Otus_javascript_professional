# Vue 3 Composition API — структура приложения

1. <b>Цель:</b>
	* Освоить Composition API и организовать структуру приложения на основе компонентов и композиционных функций.

1. <b>Описание/Пошаговая инструкция выполнения домашнего задания:</b>
	* Создать фрагмент приложения на Vue 3 с использованием Composition API. 
	* Вынести бизнес‑логику в композиционные функции и продемонстрировать повторное использование.

1. <b>Задачи:</b>
    * Создать 3–4 компонента и связать их через пропсы/эмиты.
	* Вынести логику (работа с данными/валидаторы/формы) в use* композиционные функции.
	* Добавить реактивность (ref, reactive, computed, watch).
