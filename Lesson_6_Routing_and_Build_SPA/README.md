# Клиентский роутинг и мини?SPA

1. <b>Цель:</b>
	* изучить подходы SPA и клиентский роутинг;
	* закрепить управление состоянием на уровне приложения.

1. <b>Описание/Пошаговая инструкция выполнения домашнего задания:</b>
	* Реализовать мини?SPA с клиентским роутингом и управлением состоянием. 
	* Приложение должно иметь минимум 3 страницы и единое глобальное состояние.

1. <b>Задачи:</b>
    * Настроить клиентский роутинг с минимум 3 маршрутами.
    * Реализовать глобальное состояние и использовать его в нескольких страницах.
    * Добавить форму с управлением данными через глобальный стор.
    * Обеспечить сохранение части состояния в localStorage (например, выбранные настройки).
